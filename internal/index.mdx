## What this is

This site hosts the OpenAPI reference for Dash.fi's REST APIs, for **internal staff and developers only**. It is private — access is gated to members of the Dash.fi Mintlify organisation.

The specs are **generated from source** and republished automatically when Surface is promoted to production. Do not hand-edit the files under `surface/` — they are overwritten on every publish.

## APIs

- **Surface Web API** — the BFF consumed by the Dash.fi web app.
- **Surface Admin API** — administrative/backoffice endpoints.
- **Surface Internal API** — the service-to-service (machine-to-machine) contract, consumed by other Dash.fi services such as Core.

## Download the specs

Raw OpenAPI 3.1 specs for local tooling (Postman, code generation, `openapi-diff`). Mintlify does not serve static files on authenticated sites, so these link to the source repo — use GitHub's **Download raw file** button. Requires access to `FunnelDash/api-docs`.

- [Surface Web API](https://github.com/FunnelDash/api-docs/blob/main/internal/surface/web-api.json) — BFF consumed by the web app
- [Surface Admin API](https://github.com/FunnelDash/api-docs/blob/main/internal/surface/admin-api.json) — administrative/backoffice endpoints
- [Surface Internal API](https://github.com/FunnelDash/api-docs/blob/main/internal/surface/internal-api.json) — service-to-service contract

## Freshness

Specs reflect the code at the last **production** promotion, not necessarily what is running in prod at this instant (partial deploys can lag). Treat this as the reference for the last promoted build.

## Contact

Questions about a specific API belong with its owning team. This site itself is shared engineering-docs infrastructure — see the repo `README.md` for conventions.
